
Respo SPA example
----

> Example app for https://github.com/Respo/respo

More on Respo https://github.com/Respo/respo/wiki/Quick-Start

### Usage

```bash
yarn
mkdir target/
cp index.html target/
yarn counter
yarn watch
# open localhost:8080
```

Available examples:

```bash
yarn buttons
yarn counter
yarn counter-hot
yarn shopping-cart
yarn todomvc
yarn hiccup
```

```bash
yarn watch
```
